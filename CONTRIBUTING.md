# Contributing

Thanks for your interest in contributing to Contrastly.

## Before you start

Contrastly is a small maintained project. Bug reports, small fixes, and focused improvements are welcome.

For larger changes, please open an issue first so the direction can be discussed before implementation work begins.

Contributions are reviewed on a best-effort basis, and not all suggestions, feature requests, or pull requests will be accepted. Please keep proposed changes focused, lightweight, and aligned with the project’s accessibility-oriented scope.

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run the project checks before submitting changes:

```bash
npm run check
```

## Contribution guidelines

Please try to keep changes:

- focused and easy to review
- consistent with the existing code style
- accessible and easy to use
- aligned with the project’s lightweight scope

Examples of helpful contributions:

- bug fixes
- accessibility improvements
- documentation improvements
- small UI refinements
- maintainability improvements

## Pull requests

When opening a pull request:

- explain what changed
- explain why the change was made
- include screenshots when UI behavior changes
- mention any follow-up work if relevant

Thank you for helping improve Contrastly.
